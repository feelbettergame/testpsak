# IFRS PSAK 71 UI - Vue.js Application

## 🎯 Deskripsi
Aplikasi Vue.js untuk menampilkan Standar Pelaporan Keuangan Internasional (IFRS) PSAK 71.

## ✨ Fitur
- 📊 Dashboard dengan statistik ringkasan
- 📋 Daftar standar dengan pencarian dan filter
- 🔍 Interpretasi IFRIC
- ✓ Panduan implementasi
- 📚 Materi edukasi
- 🔔 Update terbaru
- ⭐ Sistem favorit
- 🎯 Navigasi intuitif

## 🚀 Instalasi
```bash
npm install
npm run serve
```

## 🔌 API Integration
Aplikasi menggunakan Axios yang sudah pre-configured di `src/api/axios.js`
